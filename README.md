# SkillForge AI

A modern AI Learning Platform built using React JS.

## Features
- Multi-page React application
- Dark/Light mode
- Quiz system
- Skill tracker
- Responsive UI

## Tech Stack
- React JS
- Tailwind CSS
- React Router DOM

## Run Project

npm install

npm run dev
